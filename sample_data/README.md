# Sample Data for Testing EvidenceAI

## How to test:

### PDF Test
Use the provided `incident_report_sample.txt` content to create a PDF:
1. Open the text file
2. Copy content into any online "Text to PDF" converter
3. Save as `incident_report.pdf`
4. Upload to EvidenceAI

### Image Test
Search for these types of images (use royalty-free/stock images):
- "crime scene investigation training" 
- "forensic evidence markers"
- "broken window evidence photo"
- "indoor crime scene stock photo"

Or use ANY indoor photo — the BLIP model will caption whatever it sees.

### Sample Questions to Ask
After uploading, try these questions:
1. "Summarize the incident report"
2. "What evidence was found at the scene?"
3. "What time did the incident occur?"
4. "Describe the crime scene"
5. "Were there any witnesses?"
6. "What is the condition of the entry point?"
7. "Are there any inconsistencies between the report and images?"
